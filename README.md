# MeuDia+

Aplicativo desenvolvido para registro e acompanhamento da rotina diária de crianças em creches e escolas infantis.

## Funcionalidades

- Registro de entrada e saída
- Controle de alimentação
- Registro de sono
- Acompanhamento de atividades
- Comunicação com responsáveis

## Tecnologias utilizadas

- Flutter 
- Dart 
- Firebase (em implementação) 

## Objetivo

Projeto desenvolvido com foco em aprendizado e aplicação prática de desenvolvimento mobile, organização de dados e experiência do usuário.
